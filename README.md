# 🪴 mojo mojo

mojo mojo 是 bakamio 的个人博客，用来记录日常、想法、手作、旅行…以及各种稀奇古怪的小项目。也被当作她的 OCD 康复训练的实验基地。

基于开源博客主题改造，并加入自己的风格与内容结构。
