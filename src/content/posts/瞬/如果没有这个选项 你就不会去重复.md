依旧是 OCD 话题。

前阵子看到这个帖子： [İ have a hard time watching anime: OCD](https://www.reddit.com/r/OCD/comments/y9pkta/i_have_a_hard_time_watching_anime/)

讲他看动画的时候一直会重复回看，因为一直觉得 **哪里错过了** 。即使很清楚没有错过任何场景，大脑还是会强迫自己倒回去重看。所以他 **看一集要 7 个钟头** 。如果他把这一集看完了，大脑会说 **你没看完，得重新看** 。

然后评论区：

- **看直播。** 因为看直播的时候你没办法回看。然后你就会意识到你只有在「有回看这个选项」的时候才会去回看
- 所以我才不去电影院因为我不能回看…
- 和大家一起看的时候我没有这个问题，我一个人看才有这个问题
- 玩 RPG 的时候也差不多，所以我总是在一个地方跑来跑去或者进进出出同一个房间

so damn relate…（RPG 因为一句话没看清全部要重新打不就是我吗。）

我现在这个「回看」的问题好很多了，不过其他「重做」的问题依旧严重。回看好很多是因为近段时间一个人看的情况比较少，大多数都是和 cc 一起看。动画、剧集或是电影，基本上都没有太大问题。即使错过了什么或是看到精彩的片段想要倒回去，也没有很大的压力，就是很普通的事情。

但是自己一个人做的事情，比如看书、写稿、工作，我就还是有「重做」的毛病。很多时候，打开浏览器这样简单的小事，也会觉得怪怪的，要把窗口关掉重进。

尤其是自己一个人写稿，太费劲了。每当我觉得「怪怪的」，就需要 reset 当前的状态，重新开始才可以。这个 reset 的规程包括：

- quit all apps
- eject all disks
- empty the trash
- clear all clipboard
- …

弄得不好，全套来一遍，再 restart computer 才可以😢

我也意识到只有在「有这个选项」时，才会去做那些强迫行为。比如我紧张的时候总是要去清空剪贴板这件事。在一开始我不知道可以清除系统剪贴板的时候，我根本没有想要去清除它。后来知道了，就觉得剪贴板缓存着，非常不舒服，一定要清空。Hell，因为我还需要用到剪贴板历史管理（目前我用 Raycast 自带的），所以我不仅需要清空系统剪贴板，还需要清空 Raycast 剪贴板历史…

为此，我甚至做了一个 Clear System Clipboard 的 Shortcut，绑定了快捷键（非常方便按两下⇧就行），让自己随时可以清空系统剪贴板。而因为 Raycast 剪贴板历史记录清空需要好几个步骤，我曾经做过一个 macro…

是不是听上去非常复杂。我就是这么复杂地度过了一天又一天。虽然又是快捷方式又是宏，方便了不少，可是这样一来，我这个清空剪贴板的强迫行为执行次数也越来越多，越来越严重了…

所以评论区那个人说得对， **只有在「有这个选项」时，才会去做那些强迫行为** 。而越去做越不会好，就是这么一回事。

可是，没办法做强迫行为，就代表不会强迫了吗。还是有 Pure O 这样的症状呀…
